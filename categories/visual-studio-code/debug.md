### Debug

| Key    | Description             |
| ------ | ----------------------- |
| `F5`   | Start                   |
| `⇧F5`  | Stop                    |
| `⇧⌘F5` | Restart                 |
| `^F5`  | Start without debugging |
| `F9`   | Toggle breakpoint       |
| `F10`  | Step over               |
| `F11`  | Step into               |
| `⇧F11` | Step out                |
| `⇧⌘D`  | Debug sidebar           |
| `⇧⌘Y`  | Debug panel             |
