### Sidebars

| Key   | Description    |
| ----- | -------------- |
| `⌘B`  | Toggle sidebar |
| `⇧⌘E` | Explorer       |
| `⇧⌘F` | Search         |
| `⇧⌘D` | Debug          |
| `⇧⌘X` | Extensions     |
| `⇧^G` | Git (SCM)      |
