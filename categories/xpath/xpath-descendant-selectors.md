### XPath descendant selectors

| CSS                          | Xpath                                                    |
| ----                         | ----                                                     |
| `h1`                         | `//h1`                                                   |
| `div p`                      | `//div//p`                                               |
| `ul > li`                    | `//ul/li`                                                |
| `ul > li > a`                | `//ul/li/a`                                              |
| `div > *`                    | `//div/*`                                                |
| `:root`                      | `/`                                                      |
| `:root > body`               | `/body`                                                  |
